git init creates empty repository

git status shows status of repository (i.e. commits made)

git add adds file to staging area

git config --global user.email "you@example.com" updates email id

git config --global user.name "Your Name" updates name id

git commit -m "commit message" adds a commit with a message

git log shows a log of commits

git add . stages all commits and adds untracked files to staging area

git branch shows the current branch

git branch -m "branch name" changes the branch's name

git checkout -b "branch name" makes a new branch and switches to it

git checkout "branch name" switches to the branch

git merge "branch name" merges the specified branch to the current branch
